# 更新日志

## [1.0.0] - 2025-02-22

### 新增
- 支持魔搭社区 Z-Image 和 Z-Image-Turbo 模型
- 支持多种图片尺寸 (1024x1024, 1344x768 等)
- 支持自定义生成步长
- 支持每日调用次数限制
- 支持自定义提示语配置
- 异步任务处理，自动轮询结果
- Docker 环境适配

### 命令
- `画图 <描述>` - 生成图片
- `画图模型` - 查看或设置默认模型
- `画图尺寸` - 查看或设置默认尺寸
- `画图步长` - 查看或设置默认步长
- `画图状态` - 查看今日调用次数
- `画图帮助` - 查看帮助信息
